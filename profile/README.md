## BIM and Design Workflows

<a href="#">Vectorworks Mac</a> supports intelligent BIM workflows:

### Building Information Modeling

BIM in <a href="#">Vector Works for Mac</a> means your model contains not just geometry but data — intelligent
objects like walls, doors, and windows that know what they are and carry information. This lets Vectorworks
generate documentation, schedules, and analysis from the model, going far beyond simple drawing.

### Intelligent Objects

<a href="#">Vectorworks Mac</a> uses parametric, data-rich objects, so a wall is a real wall with properties,
not just lines. Changing the model updates everything derived from it, keeping the whole project coordinated
and reducing errors.
